Printing labels and values to the console can tell you what is going on in your code. It tells you when a line executes and what the value of variable are.

    Use these statements:
  • console.log("JavaScript");
  • puts "Ruby"
  • System.out.println("Java");
This is called logging. When we become more advanced coders we will write these to log files and only look at them when something is going wrong. For most applications the "user" never sees the console. So we are free to clutter it up with messages that are useful to the devvelopments team. Make your message informative and spacific.
